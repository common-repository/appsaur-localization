== Description ==
Put on your website google map with your favorite places.

"ATWPL Localizations" it helps to put on google maps with selected localizacjami and places of the world.

<strong>IMPORTANT: Back up your ftp and database before beginning the activate plugin.</strong>. Sometimes the setting of other plug-ins can cause problems working our plugin.


<h4>Features</h4>
<ul>
<li>Adding customizated google maps to page or post</li>
<li><Adding categored localizations to page or post</li>
</ul>



== Installation ==
<ol>
<li>Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your <code>wp-content/plugin</code> directory, or download from <a href="appsaur.co" title="Themes, plugins for Wordpress">appsaur.co</a>.</li>
<li>Activate the plugin through the WordPress admin panel.</li>
<li>Add map and places.</li>
<li>Add shortcode to page or post.</li>
<li>Done!</li>
</ol>
